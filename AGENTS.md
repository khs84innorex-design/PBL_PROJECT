# AGENTS.md

## 1. 프로젝트 개요
- 프로젝트명: 스마트 신호등
- 구현 언어: Python
- 목표: 차량용 신호등(빨강/노랑/초록)과 보행자 요청 버튼을 제어하는 상태 머신 기반 시스템 구현
- 대상: 임베디드 학습자용 Toy Project

---

## 2. 요구사항 요약
### 기능 요구사항
1. 차량 신호는 `초록 → 노랑 → 빨강 → 초록` 순서로 반복되어야 한다.
2. 각 LED는 잘못된 조합으로 동시에 켜지면 안 된다.
3. 각 신호는 설정된 시간 동안 유지되어야 한다.
4. 버튼 입력 시 보행자 요청이 등록되어야 한다.
5. 보행자 요청은 현재 차량 흐름이 끝난 후 빨강 구간에서 처리되어야 한다.
6. 빨강 신호 동안 보행 가능 상태를 제공해야 한다.
7. 보행자 요청 처리 후 기본 순환으로 복귀해야 한다.
8. 버튼 연타는 1회 요청으로 처리해야 한다.
9. 예기치 않은 입력이 있어도 정상 순서를 유지해야 한다.

### 비기능 요구사항
1. 신호 전환 시간 오차는 ±1초 이내
2. 버튼 입력은 1초 이내 반영
3. 전원 공급 중 반복 동작 가능해야 함
4. LED만 봐도 현재 상태를 쉽게 이해할 수 있어야 함

## 3. 개발 방향
Python으로 먼저 **상태 머신 기반 구조**를 설계한다.
하드웨어가 확정되지 않았으므로 초기 개발은 다음 2단계로 나눈다.

### 1단계: 시뮬레이션 버전
- 콘솔 출력으로 신호 상태를 확인
- 시간 흐름과 버튼 요청 처리 로직 검증
- 단위 테스트 작성

### 2단계: 하드웨어 연동 버전
- GPIO 제어 모듈 연결
- LED 출력 및 버튼 입력 반영
- 디바운싱 적용
- 실제 장치에서 반복 동작 테스트
